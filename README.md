# Chatop-Backend
Training project from Openclassroom.

---
## Desription:
Java SpringBoot Application to deliver responses for Openclassroom's Chatop Angular Frontend.

---
## Technical Stack:

- Java: 11
- Maven: 3.8.7
- Spring Boot: 2.7.12 
- Spring Security: 5.7.8
- Lombok: 1.18.26
- H2 driver: 2.1.214
- MySQL Driver: 8.0.33

---
## Starting Guide:

1. Fork git project
2. Clone repository
3. Run `mvn clean install` from root folder.
